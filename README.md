# Online Bookstore Data Management System

This project is a database system designed to manage an online bookstore using MongoDB.
The application handles data related to books, authors, customers, and orders. It uses a NoSQL approach to store and organize data efficiently, including relationships between entities through ObjectId references and embedded documents.
The system supports basic operations such as creating, reading, updating, and deleting data, as well as managing reviews and purchase history.
It also includes advanced query features such as filtering, sorting, pagination, and regular expressions for flexible data retrieval. MongoDB aggregation pipelines are used to analyze data, including customer behavior and sales statistics.

## Technologies
- MongoDB (NoSQL)
- JavaScript (MongoDB queries)

## Features
- Data management for books, authors, customers, and orders
- CRUD operations for all entities
- Embedded documents and references (ObjectId)
- Advanced search (filtering, sorting, pagination, regex)
- Data analysis using aggregation pipelines
